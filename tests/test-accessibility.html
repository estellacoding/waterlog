測試工具建議

  • 螢幕閱讀器：
    • Windows: NVDA (免費) 或 JAWS
    • Mac: VoiceOver (內建)
    • Chrome: ChromeVox 擴充功能
  • 無障礙檢查工具：
    • Chrome DevTools Lighthouse
    • axe DevTools 擴充功能
    • WAVE 評估工具
  • 鍵盤測試：
    • 只使用鍵盤（不使用滑鼠）完成所有操作
    • 確認 Tab 順序合理
    • 測試所有快捷鍵
